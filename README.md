# MidiMaster-Plugins

> **Work in progress — not currently stable.**

Community plugins for [MidiMaster](https://github.com/prgmitchell/MIDIMaster/) *(MIDI controller mapping app)*.

## Plugins

| Plugin | Description |
|--------|-------------|
| `plugin-home-assistant` | Control Home Assistant entities from MIDI controllers |
| `plugin-soundboard` | Trigger audio samples from MIDI controllers |

## Installing a Plugin

Plugins are distributed as `.midimaster` packages.

- Install from file: `Plugins -> Installed -> Install Plugin...`
- Install from the in-app Store: `Plugins -> Store`

## Status

Early development. APIs may change. Plugins may break between MidiMaster versions.
